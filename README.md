#  IHECthèque – Library Management System

IHECthèque is a C# desktop application developed as an academic project at IHEC Carthage.
The application allows efficient management of books, users, and borrowing operations.

##  Features
- Book management (add, edit, delete, search)
- User management (students, admins, librarians)
- Borrowing & return system
- Automatic book status generation
- SQL Server database with relational tables

## Technologies Used
- C#
- Windows Forms (Event-driven programming)
- SQL Server
- ADO.NET
